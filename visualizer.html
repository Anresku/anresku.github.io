DeathEmpire MMORPG

An ambitious project blending Albion Online and Isekai anime themes.

About The Project

DeathEmpire is an ambitious MMORPG for Minecraft, set in a medieval world with an Asian-inspired backdrop. The game will feature a rich history, a detailed government system, magic, levels, statistics, a player-influenced market (controlled by guilds and kingdoms), a religion system for healing magic, villages, and comprehensive trade/crafting systems.

Key Features

Player-influenced Economy
Four Unique Kingdoms
Procedural Dungeons
Deep Crafting System
Guild & Kingdom Warfare
Rich Magic System
